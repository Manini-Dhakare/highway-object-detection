# Highway Vehicle Detection

A computer vision project that detects and tracks vehicles in a highway traffic video using Python and OpenCV. The application processes each frame of the video, identifies vehicles, and displays bounding boxes with tracking IDs.

---

## 📌 Features

- Detects vehicles from a highway video
- Tracks detected vehicles across frames
- Draws bounding boxes around vehicles
- Processes video frame by frame
- Simple and beginner-friendly implementation

---

## 🛠️ Technologies Used

- Python
- OpenCV
- NumPy


## ▶️ Usage

Run the project using:

```bash
python main.py
```

The program will:
- Read the highway video
- Detect vehicles in each frame
- Track detected vehicles
- Display the processed video with bounding boxes
